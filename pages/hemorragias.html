Aplicar Presión

Usa una tela limpia o un vendaje para aplicar presión directa sobre la herida para detener el sangrado.

Elevar la Herida

Si es posible, eleva la parte del cuerpo que está sangrando por encima del nivel del corazón para ayudar a reducir el flujo sanguíneo.

Buscar Ayuda

Si el sangrado no se detiene o es severo, busca asistencia médica de inmediato.
